Why use CMAF Live Ingest:

  • A standard developed through the DASH-Industry Forum
  • An industry best practice for ingesting live media to processing entities
  • An evolution of the Azure Media Services Live Ingest specification
  • A way of ingesting CMAF media (ISO/IEC23000-19:2018)
  • A way of utilising one file format for streaming to multiple devices
